# SD card layout

Create these folders on the SD card:

```text
/roms/doom
/roms/sms
/roms/gg
/roms/gb
/roms/gbc
/retro-go/bios
```

Recommended first files:

```text
/roms/doom/doom1.wad
/roms/sms/Wonder Boy III.sms
```

Game Boy BIOS files are optional:

```text
/retro-go/bios/gb_bios.bin
/retro-go/bios/gbc_bios.bin
```

Retro-Go will create save/config folders under `/retro-go` as needed.
