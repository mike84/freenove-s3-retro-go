# Controls

The Freenove target expects an Adafruit Mini I2C Gamepad QT on the Qwiic/I2C connector.

```text
Stick       Move / menu navigation
A           Fire / confirm
B           Run / back
X           Map
Y           Weapon toggle
Start       Use
Select      Retro-Go menu / escape
```

Some cores may interpret buttons differently, but this gives Doom and the launcher a usable baseline.
